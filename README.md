# BVB Forum (Frontend)

Personal Semester Project. Conceived, designed, and delivered a full-stack community platform enabling Borussia Dortmund supporters to engage through posts, comments, live chat, and player-rating features. Provided administrators with powerful tools to manage player data and moderate community content securely.

Key highlights:

* Engineered a React front end integrated with a Java REST API, delivering a seamless and responsive user experience.
* Implemented comprehensive CRUD workflows for posts, comments, and player information to ensure efficient content management.
* Designed personalized user profiles displaying each memberâ€™s contributions, enhancing transparency and engagement.
* Built administrative interfaces to add, update, or remove player data and moderate posts/comments, ensuring data accuracy and community standards.
* Developed player performance rating and post voting systems to drive community participation and surface valuable content.
* Integrated a real-time general chat using WebSockets to enable live interaction among fans.
* Secured the backend with Role-Based Access Control (RBAC), enforcing granular permissions and protecting sensitive operations.
